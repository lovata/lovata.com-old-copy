LOVATA’s IoT Software for Healthcare

According to IDC, in the near future remote health monitoring will see the greatest investment in the healthcare industry. Check out our blog on the latest trends of the Internet of Things to find more information on the topic. LOVATA has a substantial amount of experience in implementing IoT solutions in this field.

One of the more interesting cases we’ve worked on was an IoT application for monitoring medication intake by patients and family members. We have leveraged the latest, cutting-edge modern technologies to implement the project. The solution employed intelligent blister packs which sent data upon tablet retrieval to the application. The client Application for users was built in conformance with progressive web app (PWA) standards. A control panel web application and a web app for configuring the blister packs was also created. We used the modern MEAN development stack and the IBM Bluemix cloud computing platform to implement the solution. Check out the full case study to learn more about this project.
